Confirm you are on HOST_1.

```
echo $HOST_1
ip a show eth0
```

Output information about the operating system.

```
cat /etc/os-release
```

Confirm the current kernel version.

```
uname -a
```

Output current users.

```
cat /etc/passwd
```
